Cách đơn giản đối với dùng Mac và Linux có thể cài đặt thông qua shell:

Đa dạng cách tải về khác ở thư mục lưu trữ.

Khi bạn dùng shell để cài đặt, nên gỡ cài đặt các phiên bản Rust trước đó:

curl -sSf https://static.rust-lang.org/rustup.sh | sh -s -- --uninstall

† 2 điểm đáng chú ý ABIs ở Windows: (MSVC) ABI đươc sử dụng bởi Visual Studio, và GNU ABI được sử dụng bởi GCC toolchain. Mỗi phiên bản của Rust phụ thuộc lớn vào thư viện C/C++ bạn muốn tương tác với: Visual Studio sư dụng MSVC; GNU sử dụng MinGW/MSYS2 toolchain.

MSVC yêu cầu Microsoft Visual C++ của Visual Studio 2013 or mới nhất. Cách đơn giản nhất là cài đặt Microsoft Visual C++ Build Tools 2015. Hoặc là, bạn có thể cài đặt Visual Studio 2015 hoặc Visual Studio 2013 và trong quá trình cài đặt chọn "C++ tools". Khi dùng GNU build không cần phải cài đặt thêm phần mềm nào.

Rust's đang hoàn thiện việc hỗ trợ MSVC ABI.
